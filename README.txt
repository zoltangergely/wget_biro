# wget_biro
SZTE biro és biro2 feladatainak automatizált letöltése 

wget_biro --help
biro.inf.u-szeged.hu/Hallg (vagy biro2) aktuális feladatának paraméterezhető letöltése
wget_biro "egyetlen http(s)link", vagy paraméterezve:
-s ("":biro|"2":biro2|"valami":valami)
-a azonosító
-p password
-c csoport
-L minta.zip|feladat.pdf|riport.txt link alapján
ZH vagy HF komplett letöltése
Paraméter nélkül az aktuális könyvtár soron következő számozott helyére másol
